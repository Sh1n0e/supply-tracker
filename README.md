# Supply Tracker 

## What is this?

Full-stack inventory tracker built using NEXT javascript and PostgreSQL that tracks items across different locations. In this case it is tracking between fridge, freezer and pantry (Arguably the most common places to have items that we would need to keep track of)

The grouping includes grouping, amount of stock all, gives warnings when stock is low. This is all with full CRUD functionality so that items can be added within the UI.
